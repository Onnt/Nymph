# Nymph 

## 项目简介
这个项目是我平时代码片段的一些积累  
以及一些功能的封装打包  
如果这里的东西能够帮助到你，那么我会很开心  

这里面有一些功能是需要开发者密匙的  
比如说 地址位置逆向解析，普通IP定位，彩云天气

如果你需要大量调用，请尽量换成自己的开发者密匙

## 使用说明

### 时间处理模块

* 公历和农历相互转换
1. 农历转公历
`Nym.lunar.lunarDateTo(Date lunarDate);`
2. 公历转农历 
`Nym.lunar.toLunarDate(Date date);`
